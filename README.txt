Student Management System

Technology:
- Node.js
- Express.js
- MySQL
- HTML/CSS/JavaScript

How to Run:

1. Install Node.js
2. Import database/student_management.sql into MySQL
3. Open terminal in backend folder
4. Run:
   npm install express cors mysql2 nodemon
5. Start server:
   npm run dev

Frontend:
Open frontend/index.html in browser
